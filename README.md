# Neon Mixtape Tour

# Version: 1.2    
 
# Description
A collaborative remake of one of DuckyProgramming's former games. A tower defense game based heavily off of Plants vs. Zombies 2 Neon Mixtape Tour world.

Changes cover almost all aspects of gameplay and code:    
<ul>
  <li>Graphics</li>
  <li>Plant Balancing</li>
  <li>Baseline Code (Rewritten)</li>
</ul>      

# Features
 - 29 plants
 - 31 zombies
 - 35 levels + 13 minigames + boss
 - Shop
 - Local saves

# Version 1.2 Changes:
<ul>
  <li>Plant Balancing</li>
  <li>New Plants (Spikeweed, Endurian, Stunion)</li>
  <li>More Shadow Zombie Spawns</li>
  <li>Program File Redistribution</li>
  <li>Ability to Rent Seed Slot</li>
  <li>Ability to Save Data in Local Storage</li>
  <li>Several Other Bug Fixes</li>
</ul>   
 
# How to Play
Open index.html in the NMT folder     

Pressing "Save" on the title screen will save game data to localstorage, so clearing browser cookies will clear your save data
